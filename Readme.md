
# What is Nova

> Nova is a cross plateform language
> written in C++ using gcc 6.3  
  
  ###### Paradigms:
  + Interpreted (for now)
  + Structured
  + Imperative
  + Object-oriented
  + Functional.

  ###### Example:

  ```
using nova

namespace MyProgram

class MyClass
{
    // This is a example script of Nova language.
    int main()
    {
       Nova.print("hello world");

       int var1 = 2;
       int var2 = 7;
       
       Nova.print(var1 * Nova.Math.);
    }
}
 ```
