
# What is Nova

> Nova is a cross plateform language
> written in C++
  
  ###### Paradigms:
  + Interpreted (for now)
  + Structured
  + Imperative
  + Object-oriented
  + Functional.

  ###### Example:

  ```
using nova
using nova.math

namespace myProgram

class MyClass
{
    // This code is written in Nova !
    int main()
    {
       Nova.print("hello world");

       int var1 = 2;
       int var2 = -7;
       
       Nova.print(var1 * Math.Abs(var2));
    }
}
 ```
